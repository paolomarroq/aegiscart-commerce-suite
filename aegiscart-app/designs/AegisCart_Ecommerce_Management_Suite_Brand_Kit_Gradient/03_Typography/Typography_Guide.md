# AegisCart Typography Guide

## Display / Headlines
Use **Sora SemiBold** or **Poppins SemiBold** for the AegisCart wordmark, hero titles, dashboard section headings and presentation covers.

## UI / Body
Use **Inter Regular** or **Manrope Regular** for product tables, labels, buttons, descriptions and body copy.

## Technical / Logs
Use **JetBrains Mono** for API examples, terminal commands, database snippets, log fields and Elasticsearch/Kibana references.

Recommended hierarchy:
- H1: 48–64 px / SemiBold
- H2: 32–40 px / SemiBold
- H3: 24–28 px / SemiBold
- Body: 16–18 px / Regular
- Caption: 12–14 px / Regular
