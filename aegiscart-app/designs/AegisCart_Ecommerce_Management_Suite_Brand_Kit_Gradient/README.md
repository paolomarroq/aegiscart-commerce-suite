# AegisCart Brand Kit Final

## Dirección seleccionada
**Propuesta 1 con colores de Propuesta 2**.

AegisCart se presenta como una **Ecommerce Management Suite**, es decir, software para operación, monitoreo y trazabilidad de e-commerce. No se comunica como una tienda en línea, sino como una plataforma para administrar y observar operaciones de venta digital.

## Concepto de logo
El símbolo combina tres ideas:

1. **Carrito**: comunica e-commerce de forma inmediata.
2. **Escudo**: comunica control, seguridad y confiabilidad.
3. **Líneas de movimiento**: comunican operación, velocidad y flujo digital.

## Promesa de marca
AegisCart permite gestionar productos, carritos, órdenes, clientes y eventos críticos de una operación e-commerce, con visibilidad técnica mediante monitoreo y logs.

## Eslogan recomendado
**Software para operación, monitoreo y trazabilidad de e-commerce.**

## Tipografías
- Títulos: Sora SemiBold / Poppins SemiBold
- Texto e interfaz: Inter Regular / Manrope Regular
- Datos, logs y código: JetBrains Mono

## Paleta de color
| Token | Color | Uso |
|---|---:|---|
| Midnight Navy | #07111F | Texto principal, fondos oscuros, sidebar |
| Indigo Core | #4F46E5 | Acciones principales, enlaces, gráficos |
| Violet Ops | #7C3AED | Marca, énfasis, botones principales |
| Signal Pink | #EC4899 | Alertas visuales, eventos, acentos |
| Cloud White | #F8FAFC | Fondo general |
| Slate Text | #64748B | Texto secundario |

## Uso correcto
- Usar el logo horizontal en portadas, documentos y pantallas principales.
- Usar el ícono del carrito con escudo como favicon, app icon y avatar de sistema.
- Mantener fondos claros para presentaciones y documentos.
- Usar fondo oscuro con el logo dark para dashboards o hero sections.

## Uso incorrecto
- No presentar AegisCart como una tienda en línea.
- No reemplazar el carrito por solo un escudo, porque se pierde la lectura de e-commerce.
- No usar verdes como color principal; en esta versión el sistema debe sentirse indigo/violet.
